# Assignment 10 – Data Cleaning and Preparation with Pandas

## Description
This assignment focuses on cleaning and preparing a messy customer relationship management (CRM) dataset for a retail company called UrbanStyle. The dataset contained missing values, inconsistent formats, incorrect data types, and duplicate records. Using Pandas, I transformed the raw data into a clean and structured dataset suitable for analysis, specifically for customer segmentation.

## Files Included
- README.md  
- Python script for data cleaning and transformation  
- Cleaned dataset (final output)  

## Tasks Completed
- Explored the dataset to identify data quality issues  
- Handled missing values using `.dropna()` and `.fillna()`  
- Converted data types using `.astype()`, `pd.to_numeric()`, and `pd.to_datetime()`  
- Cleaned and standardized text data using string methods  
- Removed duplicate records  
- Created new columns and derived variables for analysis  
- Applied functions using `.apply()` and `.map()`  
- Sorted and organized the dataset  
- Generated key business metrics from the cleaned data
